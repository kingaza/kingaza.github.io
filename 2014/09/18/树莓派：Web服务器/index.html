树莓派：Web服务器

By John Doe

發表於 2014-09-18

文章目錄

就性能而言，树莓派不是一个作为Web服务器的优秀选择。事实上，即使WordPress这种个人Blog系统，如果被部署在树莓派上，其访问速度也是稍显缓慢的。但另一方面，当树莓派成为一个Web服务器，用户却可以通过网页访问树莓派，比如：查看摄像头或者控制电器等等。
树莓派运行Linux操作系统。一说到Linux Web服务器，人们首先想到的是LAMP组合：Linux+Apache+MySQL+PHP。但很多人也证明了，这个组合对于树莓派来说是太重了。因此，更适合的组合应该换成nginx+Sqlite。另外有人尝试了nodejs，似乎也是一个很不错的选择。

  1. nginx
    安装nginx

    sudo apt-get install nginx

    启动nginx

    sudo /etc/init.d/nginx start

    nginx的www根目录默认为/usr/share/nginx/www

  2. nodejs
    安装也很简单

    curl -O www.nodejs.org/dist/v0.10.29/node-v0.10.29.tar.gz
tar -zxvf node-v0.10.29.tar.gz
cd node-v0.10.29
./configure
make
make install

    不过得吐槽一下，这个make过程太漫长了，足足用了四个小时。。。orz
    然后顺手安装了hexo，一个静态页面的blog系统。访问速度嘛，只能说还凑合。。。
    我做这件事情，主要也是为了试验访问树莓派中的图像数据的情况。结果比较乐观。
    国外大牛就高大上了：一台树莓派+一个DeviantArt账号=一面动态精美插画墙
    DeviantArt.com是一个插图社区，Wiebe写了一个脚本，将网站内的图片导入到树莓派内，并且通过显示屏，动态得将图片展示到一面墙上。真服了。
