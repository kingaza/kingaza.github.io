树莓派：网络配置

By John Doe

發表於 2014-09-18

文章目錄

  1. ssh
    树莓派官方系统Raspbian自2012-9-18版本之后，均设置了ssh服务随系统自动启动。因此只需要一个ssh客户端就可以远程登陆raspberry pi了。这里推荐的是putty

  2. ftp
    安装vsftpd服务器

    sudo apt-get install vsftpd

    启动ftp服务

    sudo service vsftpd start

    编辑vsftpd的配置文件

    sudo nano /etc/vsftpd.conf

    设置后保存退出

    anonymous_enable=NO
local_enable=YES
write_enable=YES
local_umask=022

    重启vsftpd服务

    sudo service vsftpd restart

    以用户名pi登录，密码是raspberry。ftp的根目录是/home/pi，即pi用户的HOME目录。测试成功。

  3. xrdp
    需要在raspbain下面安装一个服务xdrp，运行命令

    sudo apt-get install xrdp

    然后就可以使用windows的远程桌面连接了。输入Raspberry Pi的地址


    输入username和password，就可以进入系统
