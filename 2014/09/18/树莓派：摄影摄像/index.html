树莓派：摄影摄像

  • 树莓派

本来家里有个放了很久的网络摄像头，还打算用来尝试一下图像拍摄。可是不知道什么时候已经不工作了，山寨产品果然不可靠。然后在elinux的RPi USB Webcams list里面翻了翻，貌似罗技的摄像头支持度还不错。比较过后决定C270，评价挺不错：”image/video quality is crisp”，虽然不知道用的是什么镜头跟芯片——网络摄像头也不会是什么好东西了。然后有人说还需要带电源的USB Hub，就顺手在京东把这两个东西都顺下来了。昨晚下的单，快递哥哥今天早上就到了，赞一个。
晚上散步回来，立马插上C270，启动树莓派。

咦，居然立马识别出来了，还没用带源的USB Hub。。。白担心了。。。 =_=!!!
再看一下

嗯，挂在video0了。树莓派（或者Linux）上用来抓图的软件中，最简单的应该是fwebcam了。先用它做个简单的尝试，直接把video0的图像抓下来就好：

pi@raspberrypi ~ $ sudo fswebcam -d /dev/video0 -r 640x480 --bottom-banner --title "Raspberry Pi @xiaoqiang" --no-timestamp /home/pi/LogitechC270.jpg
--- Opening /dev/video0...
Trying source module v4l2...
/dev/video0 opened.
No input was specified, using the first.
--- Capturing frame...
Corrupt JPEG data: 1 extraneous bytes before marker 0xd2
Captured frame in 0.00 seconds.
--- Processing captured image...
Putting banner at the bottom.
Setting title "Raspberry Pi @xiaoqiang".
Clearing timestamp.
Writing JPEG image to '/home/pi/LogitechC270.jpg'.

这也太顺利了吧。。。
更高级一点的应用，则是用motion甚至opencv来获取处理图像视频。motion非常强大，可以监测画面变化；而OpenCV更加不用说了。不过OpenCV在树莓派上跑还是有点儿吃力，据说CPU基本保持在6、70%左右。我觉得下一代的树莓派会更值得期待。
